## Установка проекта
1. Установите необходимые [зависимости](#зависимости)
2. Клонируйте репозиторий с кодом проекта:
   ```bash
   git clone https://github.com/M1nutka/litevls.git
    ```
3. Настройте базу данных

    * Используя командную консоль, перейдите в корень проекта и выполните команду:
    bash

    * ./mvnw spring-boot:run

        (Проект использует Maven Wrapper)

    * API будет доступен по адресу http://localhost:8080

## Зависимости
### Обязательно

* Java Development Kit 25 – набор инструментов для разработки на Java

* PostgreSQL – реляционная база данных

### Опционально

* Docker – для запуска тестов с Testcontainers

* Postman – для тестирования API (альтернатива: curl)

## Подключение базы данных

### Проект использует PostgreSQL.

__Для создания базы данных:__

* Скачайте и установите PostgreSQL

* Создайте пустую БД с названием myproject



Параметры подключения к БД настраиваются в файле src/main/resources/application-local.properties (или application.properties):

    Хост: localhost

    Порт: 5432

    База данных: myproject

    Пользователь: postgres

    Пароль: ВАШ_ПАРОЛЬ

## Документация API

__Для взаимодействия с API используйте HTTP-клиенты (curl, Postman).__

Предполагаемая структура эндпоинтов (основана на коде в репозитории, например, TransportationService):

### Основные методы

**Пользователи (эндпоинты):**
| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| POST | /users/register | Регистрация нового пользователя | Все |
| POST | /users/login | Авторизация пользователя | Все |
| GET | /users/admin/users | Получение списка всех пользователей | ADMIN |
| GET | /users/admin/users/{id} | Получение пользователя по ID | ADMIN |

**Транспортные заявки (эндпоинты):**
| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| GET | /transportation/admin/all | Получение всех транспортных заявок | ADMIN |
| GET | /transportation/user/{id} | Получение заявки по ID | USER |
| GET | /transportation/user | Получение всех заявок текущего пользователя | USER |
| POST | /transportation/user | Создание новой транспортной заявки | USER |
| PUT | /transportation/user/{id} | Обновление существующей заявки | USER |
| DELETE | /transportation/user/{id} | Отмена заявки пользователем | USER |
| DELETE | /transportation/admin/{id} | Отмена заявки администратором | ADMIN |
| POST | /transportation/admin/approve/{id} | Одобрение заявки администратором | ADMIN |

__Пользователи:__
| Атрибут | Тип | Описание |
|---------|-----|----------|
| id | Long | Первичный ключ |
| name | String | Имя пользователя |
| lastname | String | Фамилия пользователя |
| phone | String | Телефон |
| email | String | Адрес электронной почты |
| password | String | Хэш пароля |
| Role | USER or ADMIN | Роль |
| active | boolean | Активность |

__Транспортные операции:__

| Атрибут | Тип | Описание |
|---------|-----|----------|
| id | Long | Первичный ключ |
| user | UserEntity | Пользователь(вторичный ключ) |
| date | LocalDate | День в который должны забрать груз |
| typeCargo | enum: Tr.TypeCargo | Тип груза |
| lenght | Double | Длинна |
| width | String | Ширина |
| height | Integer | Высота |
| placeOfDeparture | String | Пункт отправления |
| deliveryAddress | String | Адрес доставки |
| status | enum: Tr.Status | Статус |
